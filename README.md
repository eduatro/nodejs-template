# nodejs-template
Is a back end template in nodejs with some middlewares and modules ready to use.

## Main nodejs modules
- expressjs
- moongose

## Middlewares

- express-jwt
- request log

## Commons
- cors option
- token generator
- logger

## Main file structure
- config
- log
- src
  - commons
  - components
  - middlewares

## Includes
 - .eslintrc.js
 - config
  - config.js

## How Use
Run `npm install`
Run `npm start`
